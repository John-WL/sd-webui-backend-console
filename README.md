# sd-webui-backend-console
A1111 extension to add the console in the UI. 

![image](https://github.com/John-WL/sd-webui-backend-console/assets/34081873/2b051eed-1024-4f73-96e8-660e4afc31f2)

## Features 
- Chose where to display the console in the UI
- Url setting for remote backends

> Note: you need the `--api` flag for this extension to work
